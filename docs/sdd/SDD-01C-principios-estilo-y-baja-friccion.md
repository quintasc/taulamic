# SDD-01C - Principios de estilo y baja friccion

## 1) Objetivo

Definir como debe sentirse y verse la aplicacion para que sea amigable, elegante y facil de usar desde el primer contacto.

## 2) Benchmark de referencia

Se toman como referencia de mercado:

- PerfectTablePlan
- Planning Pod
- Prismm

Uso del benchmark:

- inspirar buenas practicas de flujo,
- comparar claridad de interaccion,
- evitar copiar diseno o contenido de forma literal.

## 3) Direccion visual (UI)

La interfaz debe ser:

- sobria,
- elegante,
- poco cargada,
- relajante,
- alegre sin distracciones.

Traduccion operativa:

- uso moderado de color de acento,
- espacios en blanco generosos,
- tipografia legible y consistente,
- jerarquia visual clara por tamano/peso,
- iconografia simple y comprensible.

## 4) Principios de experiencia (UX)

- **Baja friccion:** menos pasos, menos esfuerzo mental.
- **Guiado progresivo:** mostrar solo lo necesario en cada etapa.
- **Acciones obvias:** el siguiente paso debe ser evidente.
- **Feedback inmediato:** cada accion confirma resultado rapido.
- **Lenguaje simple:** microcopy no tecnico.

## 5) Objetivos de usabilidad (MVP)

- Configuracion inicial de evento (admin) en <= 10 minutos.
- Importacion de plano y correccion manual en <= 5 minutos objetivo.
- Precarga de 500 invitados por Excel (con validacion y confirmacion) en <= 10 minutos objetivo.
- Envio de invitaciones masivo (150 invitados) en <= 3 minutos.
- Respuesta RSVP de invitado en <= 45 segundos.
- Comparacion de candidatas Top-3 en <= 2 minutos por admin novato.

## 6) Requisitos visuales minimos para Figma

- Sistema de color con roles semanticos (info, exito, alerta, error).
- Escala tipografica definida.
- Componentes base: botones, inputs, chips de estado, tarjetas, tablas/listas.
- Variantes para estados de carga, vacio y error.
- Modo responsive para escritorio y movil.

## 7) Comentarios para principiantes

- **Baja friccion:** cuando el usuario consigue su objetivo con poco esfuerzo.
- **Microcopy:** textos cortos de interfaz (botones, mensajes, ayudas).
- **Jerarquia visual:** forma de guiar la mirada hacia lo mas importante.
