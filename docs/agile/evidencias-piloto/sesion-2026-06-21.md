## Feedback PO (decisiones post-piloto)

Documento canónico: **`docs/ux/spec-invitados-panel-v2-post-piloto.md`**

### 1. Configuración — ubicación Google Maps

- Añadir posibilidad de indicar ubicación con **Google Maps** en configuración del evento.
- Utilidad: enlazar en invitaciones por correo (HU-10).
- **Piloto actual:** campo texto `location` en meta local.

### 2. Separación estricta Invitados vs Afinidades

| Pestaña | Rol |
|---------|-----|
| **Invitados** | Base de datos logística: nombres, RSVP, categorías, necesidades físicas (intolerancias, movilidad) |
| **Afinidades** | Motor de reglas: agrupación global, afinidades/incompatibilidades interpersonales |

Alineado con ADR-018; esta sesión **refuerza** la separación con spec de UI detallada.

### 3. Rediseño pantalla Invitados (v2)

- Tabla: `[ ]` | RSVP | Nombre | Correo | Teléfono | Categoría | Alertas 🌾/♿ | Invitación | ⋮
- **+ Añadir invitado** (coral, arriba derecha) → **drawer lateral** para detalle profundo
- Filtros: buscador + chips («Solo menú especial», «Pendientes de confirmar», …)

### 4. Acciones masivas — Bulk Action Bar

- Activación por checkboxes
- Barra flotante: «N invitados seleccionados»
- Acciones: Asignar categoría, Cambiar estado, Enviar invitaciones, Eliminar (+ exportar / plantilla según spec)
